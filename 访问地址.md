## 📋 Expo 快速步骤

### 第一步：打开 Expo Snack
在电脑浏览器访问：
```
https://snack.expo.dev
```

### 第二步：粘贴代码
删除编辑器里的默认代码，粘贴以下代码：

```javascript
import React from 'react';
import { WebView } from 'react-native-webview';
import { SafeAreaView, StyleSheet, ActivityIndicator, Text, View } from 'react-native';

export default function App() {
  return (
    <SafeAreaView style={styles.container}>
      <WebView
        source={{ uri: 'https://gas-liquid-app.loca.lt' }}
        style={styles.webview}
        startInLoadingState={true}
        renderLoading={() => (
          <View style={styles.loading}>
            <ActivityIndicator size="large" color="#1890ff" />
            <Text style={styles.loadingText}>加载中...</Text>
          </View>
        )}
        javaScriptEnabled={true}
        domStorageEnabled={true}
        originWhitelist={['*']}
        mixedContentMode="always"
      />
    </SafeAreaView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  webview: {
    flex: 1,
  },
  loading: {
    position: 'absolute',
    top: 0,
    left: 0,
    right: 0,
    bottom: 0,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#fff',
  },
  loadingText: {
    marginTop: 10,
    fontSize: 16,
    color: '#666',
  },
});
```

### 第三步：安装依赖
1. 点击顶部 `Libraries` 按钮
2. 搜索 `react-native-webview`
3. 点击 `Install` 安装

### 第四步：构建 APK
1. 点击顶部的 **↓** 或 **Export** 按钮
2. 选择 **Build for Android**
3. 等待构建完成（3-10 分钟）
4. 点击 **Download APK** 下载

### 第五步：安装到手机
1. 把 APK 传到手机
2. 点击安装（需要开启"允许安装未知来源应用"）
3. 安装完成后打开应用

## ❓ 常见问题

### Q: 验证页面过不去怎么办？
**A:** 刷新页面，或者换一个浏览器试试

### Q: 应用打不开/白屏？
**A:** 
- 检查网络连接
- 确保已完成首次验证
- 清除浏览器缓存重试

### Q: 需要长期使用的地址？
**A:** localtunnel 是临时的，每次重启服务地址会变。如需长期使用，建议：
- 使用固定域名服务器
- 或者生成 APK 离线使用
