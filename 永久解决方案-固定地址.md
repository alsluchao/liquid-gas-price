# 🌐 永久解决方案 - 使用固定访问地址

## 问题根源

云服务器的公网 IP 是动态分配的，可能会在以下情况变化：
- 服务器重启
- 网络环境变化
- 服务商维护

**每次 IP 变化都需要重新构建 APK，这非常麻烦！**

---

## ✅ 推荐方案：使用隧道服务

### 方案对比

| 方案 | 稳定性 | 成本 | 易用性 | 推荐度 |
|------|--------|------|--------|--------|
| ngrok | ⭐⭐⭐⭐ | 免费/付费 | 简单 | ⭐⭐⭐⭐ |
| cloudflare tunnel | ⭐⭐⭐⭐⭐ | 免费 | 中等 | ⭐⭐⭐⭐⭐ |
| 固定公网 IP | ⭐⭐⭐⭐⭐ | 付费 | 简单 | ⭐⭐⭐ |
| 域名 + DDNS | ⭐⭐⭐⭐ | 付费 | 复杂 | ⭐⭐⭐ |

---

## 🚀 方案一：Cloudflare Tunnel（免费，推荐）

### 优势
- ✅ 完全免费
- ✅ 固定域名
- ✅ 不需要公网 IP
- ✅ 自动 HTTPS

### 步骤

1. **注册 Cloudflare 账号**
   - 访问：https://dash.cloudflare.com/sign-up

2. **添加域名**（可使用免费域名）
   - 购买域名或使用 Freenom 免费域名

3. **安装 Cloudflared**
```bash
# 下载并安装
curl -L https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 -o cloudflared
chmod +x cloudflared
sudo mv cloudflared /usr/local/bin/
```

4. **登录认证**
```bash
cloudflared tunnel login
```

5. **创建隧道**
```bash
cloudflared tunnel create liquid-gas-app
```

6. **配置隧道**
```bash
# 创建配置文件
mkdir -p ~/.cloudflared
cat > ~/.cloudflared/config.yml << EOF
tunnel: <your-tunnel-id>
ingress:
  - hostname: liquid-gas.your-domain.com
    service: http://localhost:5000
  - service: http_status:404
EOF
```

7. **添加 DNS 记录**
```bash
cloudflared tunnel route dns liquid-gas-app liquid-gas.your-domain.com
```

8. **启动隧道**
```bash
cloudflared tunnel run liquid-gas-app
```

## 🚀 方案二：ngrok（快速简单）

### 优势
- ✅ 5 分钟内完成
- ✅ 免费版可用
- ✅ 不需要域名

### 步骤

1. **注册 ngrok 账号**
   - 访问：https://dashboard.ngrok.com/signup

2. **获取 authtoken**
   - 登录后在 Dashboard 找到 authtoken

3. **安装 ngrok**
```bash
# 下载
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
tar -xzf ngrok-v3-stable-linux-amd64.tgz
sudo mv ngrok /usr/local/bin/
```

4. **配置认证**
```bash
ngrok config add-authtoken <your-authtoken>
```

5. **启动隧道**
```bash
ngrok http 5000
```

6. **获取固定域名**（需要付费）
   - 免费版每次启动域名会变
   - 付费版可以固定域名

## 🚀 方案三：购买固定公网 IP

### 优势
- ✅ 最稳定
- ✅ 不需要额外配置

### 步骤
1. 联系云服务商（阿里云/腾讯云）
2. 申请弹性公网 IP（EIP）
3. 绑定到服务器
4. 费用约 20-50 元/月

---

## 📋 更新 APK 的步骤

使用隧道服务后，更新 APK：

1. **修改 `expo-apk-project/App.js`**：
```javascript
source={{ uri: 'https://liquid-gas.your-domain.com' }}
```

2. **重新构建 APK**：
```bash
cd expo-apk-project
eas build --platform android --profile preview
```

---

## 💡 建议

**短期方案**：先手动重新构建 APK v1.0.3
**长期方案**：配置 Cloudflare Tunnel 或购买固定 IP

这样可以避免每次 IP 变化都要重新构建 APK！
