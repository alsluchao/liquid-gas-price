# HBuilderX APK 打包完整教程

## 为什么选择 HBuilderX？

✅ **数据安全**：数据存储在 Supabase 云端，不会丢失
✅ **长期可用**：支持热更新，修改功能无需重新打包
✅ **简单快速**：1-2 小时完成，无需配置开发环境
✅ **完全免费**：云打包免费，无服务器费用
✅ **小范围分发**：直接发 APK 给别人安装即可

---

## 第一步：安装 HBuilderX（10分钟）

### 1. 下载安装
- 官网：https://www.dcloud.io/hbuilderx.html
- 下载 **正式版**（App 开发版）
- 解压即可使用（Windows/Mac 都支持）

### 2. 登录账号
- 打开 HBuilderX
- 点击右上角登录/注册
- 注册 DCloud 账号（免费）

---

## 第二步：创建 uni-app 项目（5分钟）

### 1. 新建项目
```
菜单 → 文件 → 新建 → 项目
```

### 2. 选择模板
- 选择 **uni-app**
- 项目名称：`液气进出管理`
- 模板：**默认模板**
- 点击创建

### 方案 B：WebView 模式（最快，5分钟）

直接用 WebView 加载你的 H5 应用：

```vue
<!-- pages/index/index.vue -->
<template>
  <web-view :src="h5Url"></web-view>
</template>

<script>
export default {
  data() {
    return {
      h5Url: 'https://你的域名.com' // 替换为你的 H5 地址
    }
  }
}
</script>
```

**优势**：
- ✅ 代码完全不用改
- ✅ 更新 H5 后 APK 自动同步
- ❌ 需要先部署 H5 到公网

## 第四步：配置应用信息（10分钟）

### 1. 打开 manifest.json

```json
{
  "name": "液气进出管理",
  "appid": "__UNI__XXXXXXX",
  "description": "液气进出价格统计系统",
  "versionName": "1.0.0",
  "versionCode": "100",
  "app-plus": {
    "android": {
      "permissions": [
        "<uses-permission android:name=\"android.permission.INTERNET\"/>"
      ]
    }
  }
}
```

### 2. 配置 App 图标
- 点击 manifest.json 的 **App 图标**
- 上传一张图片（建议 512x512）
- 自动生成各尺寸图标

---

## 第五步：云打包 APK（10分钟）

### 1. 开始打包
```
菜单 → 发行 → 原生App-云打包
```

### 2. 选择配置
- **Android**：勾选
- **iOS**：不勾选（需要开发者账号）
- **证书**：使用 DCloud 公共证书（测试用）
- 点击 **打包**

### 3. 等待完成
- 打包时间：5-10 分钟
- 完成后自动下载 APK

## 常见问题

### Q1：数据会丢失吗？
**A：不会**。数据存储在 Supabase 云数据库，即使删除 APK，数据也在云端。

### Q2：Supabase 收费吗？
**A：有免费额度**。每月 500MB 数据库 + 2GB 文件存储，小范围使用完全够用。

### Q3：APK 会过期吗？
**A：不会**。安装后永久可用，除非你主动下线 Supabase 项目。

### Q4：如何更新功能？
**A：两种方式**
- WebView 模式：更新 H5 代码即可，APK 无需重装
- 原生模式：重新打包 APK 发给别人

### Q5：能上架应用商店吗？
**A：可以**，但需要：
- 软件著作权
- ICP 备案
- 应用商店审核
- 小范围使用不建议
