# 🎯 Expo Snack 完整操作步骤（新版界面）

## 📍 第一步：找到菜单按钮

### 顶部菜单栏位置

**从左到右通常有：**

1. **Project**（最左侧）
   - 点击后可以看到项目设置
   - 可能有 `Export` 或 `Download` 选项

2. **中间区域**
   - 可能显示项目名称
   - 或者有一些图标按钮

3. **右侧区域**（重点！）
   - **Run on device** - 在手机上运行
   - **Export** 或 **↓** 图标 - 导出/构建 ✅ **找这个！**
   - **Share** - 分享
   - **设置图标**（齿轮形状）

---

## 🔍 找不到 Libraries 的替代方法

### 方法 1：直接使用内置 WebView

**好消息**：`react-native-webview` 可能已经包含在 Expo SDK 中！

**测试方法**：
1. 看右侧预览区域是否有报错
2. 如果没有红色错误提示，说明依赖已存在
3. 直接进入下一步构建 APK

---

## 🚀 第二步：构建 APK

### 方式 1：点击 Export 按钮
1. 找到页面右上角的 **`Export`** 按钮或 **`↓`** 图标
2. 点击后会弹出选项
3. 选择 **`Build for Android`** 或 **`Download Project`**

### 方式 2：点击 Project 菜单
1. 点击页面左上角的 **`Project`** 按钮
2. 在下拉菜单中找 **`Export`** 或 **`Build`**
3. 选择 Android 平台

### 方式 3：使用快捷键
- 按 `Ctrl + E`（Windows）或 `Cmd + E`（Mac）
- 或者按 `Ctrl + Shift + P` 打开命令面板
- 搜索 "export" 或 "build"

---

## 📱 如果找不到构建按钮

### 替代方案：下载项目代码

1. 找到 **`Export`** 或 **`Download`** 按钮
2. 选择 **`Download Project`** 或 **`Download ZIP`**
3. 解压下载的 ZIP 文件
4. 发给我，我帮你用其他方式构建 APK
