## ⚠️ 注意事项

### 关于 IP 地址变化

云服务器的公网 IP 可能会在以下情况变化：
- 服务器重启
- 网络环境变化
- 服务商维护

**解决方案**：
1. **短期**：IP 变化时在 APP 内更新地址（本版本支持）
2. **长期**：申请固定 IP 或使用域名（需额外配置）

### 如何获取最新 IP 地址

如果无法连接，请按以下步骤获取最新 IP：

**方法一：查看项目文档**
- 检查项目中的 `当前服务器地址.txt` 文件

**方法二：命令行查询**
```bash
curl ifconfig.me
```

## 💡 常见问题

### Q1: 忘记了服务器地址怎么办？

**A**: 联系管理员获取最新地址，或在项目文档中查看。

### Q2: 输入地址后无法连接？

**A**: 检查以下几点：
1. 地址格式是否正确（必须以 http:// 或 https:// 开头）
2. 服务器是否正常运行
3. IP 地址是否最新
4. 网络连接是否正常

### Q3: 如何恢复默认配置？

**A**: 点击"设置"按钮，重新输入地址即可。

### Q4: 卸载重装后需要重新配置吗？

**A**: 是的，卸载 APP 会清除本地数据，需要重新输入地址。
